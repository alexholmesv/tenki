== README

TENKI V.0.1

This is a simple Rails weather app that shows the current temperature and forecast for 5 different cities: Santiago, Bangkok, Bamako, Paris and New York.

* Ruby version
- 2.1.5

* Additional Gems
- Barometer
- Rails_12factor
- PG for production in Heroku

* CSS
- Bootstrap CDN

* Expected changes on next version
- AJAXed views to reduce latency on all city pages using jquery.smoothState
